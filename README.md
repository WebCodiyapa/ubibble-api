# My Flask App

A simple Flask Blog App with Authorization
integrated docker

## To run the app:
### Requirements:
    - Docker

### To run in production with docker:
```bash
    $ git clone https://github.com/alaturqua/my_flask_app.git .
    $ docker-compose -up
```

### To run locally in development:
```bash
    $ git clone https://github.com/alaturqua/my_flask_app.git .
    $ pip3 install -r requirements.txt
    $ python3 app.py
```
